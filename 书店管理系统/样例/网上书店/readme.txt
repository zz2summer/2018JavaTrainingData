
配置工作：
（1）关于eclipse
1.下载eclipse
http://www.eclipse.org/downloads/（下载第二个）
2.数据库（推荐使用Mysql,我共享的文件用的是mysql,每个数据库连接的方法不一样，用到的包也不一样)
数据库驱动包（mysql-connector-java-5.0.4-bin）这个是放到WEB-INF的lib文件夹下
3.tomcat服务器（jbuilder是自带了tomcat服务器，但是jbuilder用的很纠结，建议自己下一个tomcat）
http://tomcat.apache.org/
配置tomcat的环境可以请教一下别人
（2）关于Jbulider
我共享的文件里面有一个用jbuilder写的，其实jbulider原理和eclipse基本差不多，好像我的jbuilder在我这运行的很纠结，所以我不保证我的那个jbuilder程序的正确性（不推荐大家用jbuilder）
关于连接数据库的话原理差不多
大家可以看到共享文件里面有三个版本的eclipse写的网上书店
版本1：基本上是和书上一样的，大家都知道书上的程序是不完整的，
注释:由于我是做的很快，里面有一点逻辑错误，我也没找出来，大家可以看看
版本2：之所以写了这么几个版本是想给大家一个提示，自己可以在上面添加功能，完善这个程序，这个程序里就添加了一个注册的功能，希望大家可以自己添加自己的功能
版本3：最开始我没有敲书上的代码是因为，书上的网上书店实在是太简陋了，第三个版本中我就是随便找了个网页模板，意思是大家可以设计一下自己的网上书店，把界面做的好看一点(如果不会写网页的话，你可以把你想要做出网页的效果设计出来，然后把图片资料发给我我可以帮大家做自己的网页，做这个工作主要是为了让老师检查的时候看到的都是不一样的）
版本。。。（由于时间原因我只弄了三个版本，如果还想把自己的网站做的更复杂的话，我这之前也做了一些）
